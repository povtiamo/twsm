# TWSM接口自动化测试脚本
--本程序仅支持Windows，需要其他操作系统运行环境请联系作者

--config存放配置文件目录

--data存放读取数据文件目录

--docs存放安装环境和说明文档目录

--resource存放资源图片附件等目录

--utils存放脚本源码目录

### 一、准备环境：
1.打开docs文件夹，根据安装手册README.md准备运行环境

### 二、配置文件conf.ini填写规范：
1.打开config文件夹，根据配置手册README.md准备配置环境

### 三、运行脚本：
1.环境准备好，配置填写完成后，运行start.bat"即可
